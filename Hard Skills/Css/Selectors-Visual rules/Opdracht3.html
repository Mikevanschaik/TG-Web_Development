Type selector

De type selector (of element selector) wordt gebruikt om snel alle elementen van hetzelfde type te stylen zoals alle p elementen
p {color: white}

Class selector

De class selector selecteert elementen met een specifiek class attribuut. Om elementen met een specifieke klasse te selecteren, schrijf je een punt (.) teken, gevolgd door de naam van de klasse.
.intro { background-color: red; }

Descendant selector

De Descendant selector gebruiken we om elementen te stylen die "afstammen" van andere elementen, zo kun je bijvoorbeeld alle p elementen binnen een bepaalde div aanpassen
p { background-color: lavender; }
