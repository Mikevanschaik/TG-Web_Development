Child Selectors

Een child selector wordt gebruikt om alleen directe "afstammelingen" te selecteren, en ziet er zo uit ol > li {color:purple:} . Het voordeel van een child selector is dat het mogelijk is elementen te selecteren die een parent > child relatie hebben.

Adjacent Selectors

Een adjecent selector wordt gebruikt als een manier om twee selectors te combineren. Adjecent betekent aangrenzend, je kunt met een adjecent selector dus het element stylen dat direct grenst aan je eerste selector, je gebruikt hiervoor het + symbool
div + p { color : purple; }

General Selectors

De general selector wordt gebruikt om twee selectoren te scheiden en past stylinh toe op elk versie van het tweede element, dat in het eerste element voorkomt. een voorbeeld hiervan:
p ~ span { color : purple; }
Hiermee kleur je all span elementen in een paragraaf.
