Opdracht 1

Position Static:

Static position is de standaardinstelling. Wanneer een element static is hebben de top right bottom left en z-index waarden geen effect. Een static element wordt gepositioneerd volgens de normale stroom van het document.

Position Relative:

Wanneer je een element position op relative zet, zonder de volgende waarden te gebruiken: top right bottom left gebeurt er niets. Wanneer je een extra position toevoegt, zoals left : 20px; zal het element 20px naar rechts verplaatsen vanaf zijn normale positie. Een relative element wordt gepositioneerd volgens de normale stroom van het document.

Position Absolute:

Wanneer een element een position heeft van absolute dan wordt het element uit de documentstroom gehaald. Dit betekent dat het geen ruimte meer in beslag neemt zoals static en relative. Wanneer de position absolute is van een element dan wordt het element gepositioneerd met verwijzing naar de closest parent element die zijn position of relative heeft staan.

Position Fixed:

Wanneer een element een position heeft van fixed, wordt het element relative gepositioneerd ten opzichte van de viewport. Wat dit betekent is dat het altijd op dezelfde plaats blijft staan, zelfs als de pagina wordt gescrold. De waarden van top right bottom left worden gebruikt om het element te positioneren.Een fixed element laat geen gat in de pagina achter waar het element normaal gesproken zou hebben gestaan.

Position Sticky:

Sticky position is een hybride van relative en fixed positioning. Het element met een sticky position wordt behandeld alsof het een relative position heeft totdat het een gespecificeerde drempel overschrijdt, waarna het wordt behandeld als een fixed position.
