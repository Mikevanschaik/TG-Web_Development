Cascade

Cascade betekent dat de volgorde van de CSS-regels ertoe doet; CSS wordt gelezen en uigevoerd van boven naar beneden. Wanneer je twee keer dezelfde styling geeft aan hetzelfde element wordt dus de onderste styling regel toegepast.
h1 { color: black; } , h1 { color: purple; } in dit voorbeeld kleurt de h1 header dus paars.

Wat is Inheritance?

Inheritance betekent erfenis, dit houdt in het child-element "erft" sommige styling van het parent-element. Bijvoorbeeld, als je een element een kleur en een font-family property geeft zal alle inhoud van het element worden gestyled. Tenzij je die specifieke content een andere kleur en een andere font-family hebt gegeven.
Sommige properties, zoals width werken niet met inheritance.
