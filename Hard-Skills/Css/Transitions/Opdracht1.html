Opdracht 1

Wat zijn transitions?

Transitions zijn de makkelijkste manier om animaties toe te voegen aan een website. Met transitions defineer je de overgang tussen twee elementen.


Welke properties zijn er?

transition-property:

Specificeert de naam of namen van de CSS properties die geanimeerd moeten worden. Alleen de properties die worden vermeld worden geanimeerd.

transition-duration:

Specificeert de duratie van een animatie

transition-timing-function:

Dit zorgt ervoor dat alle animaties precies de zelfde tijd nodig hebben om klaar te zijn.

transition-delay:

Definieert hoe lang een property moet wachten voordat de transition begint.
